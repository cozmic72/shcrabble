About Shcrabble

Shcrabble is a multiplayer Scrabble game for the Shavian alphabet.

Dictionary: Readlex

Credits

This digital game is based on the physical Shavian Scrabble board designed by Discord user @brentusfirmus.

Changelog

Version 1.2.0 (Current)
  • Rotation tile mode: 15 rotationally symmetric letter pairs share flippable tiles, with simulation-calibrated point values
  • Extended alphabet option: yea and oevre replace err and air as rare high-value tiles
  • AI bot opponents: four difficulty levels (Beginner, Casual, Intermediate, Expert) with rack management and endgame strategy
  • Live opponent tile preview: see tiles being placed in real-time before submission
  • Click any player to highlight their tiles on the board
  • Per-mode custom tile distribution editor with rotated point values
  • Improved mobile pinch-zoom clipping
Version 1.1.0
  • Mobile support: touch drag/drop, pinch-zoom board, portrait layout
  • Endgame mechanics: casual & tournament scoring, auto-end when bag empty
  • Game configuration: customizable rules, rack size, tile distribution, voting
  • Ownership transfer when owner leaves
  • Improved dialogs: config summaries, always show for owner
  • Bug fixes: null game state handling, spectator reconnection
Version 1.0.0
  • Initial release
  • Core multiplayer Scrabble gameplay
  • Shavian alphabet support
  • Real-time word validation
  • Voting on disputed words
  • Spectator mode
